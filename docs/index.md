# Winston HPC User Guide

Welcome to Winston — a small HPC cluster for running CPU and GPU workloads via Slurm.

Use the navigation on the left to get started:

- **Accessing Winston** (SSH + keys + VPN)
- **Running jobs** (batch, interactive, GPU)
- **Managing software** (Spack + Miniconda)
- **Moving data** (scp/rsync + GUI tools)

!!! tip "New users"
    If you’re not sure where to start: read **Getting started → Accessing Winston**, then **Slurm jobs → Overview**. If you have not had an account created yet, please email Tom @ [t.robinson7@lse.ac.uk](mailto:t.robinson7@lse.ac.uk) to gain access.
