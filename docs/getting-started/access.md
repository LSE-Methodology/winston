# Accessing Winston

Winston is accessible via SSH from within the LSE network.

## SSH from within the LSE network

```bash
ssh user@ip.address
```

!!! note
    Replace `user` and `ip.address` with your Winston username and the IP address provided when your account was created.

## Access from outside LSE (VPN)

To access Winston off-campus, connect to the [LSE VPN](https://info.lse.ac.uk/staff/divisions/dts/help/guides-faqs/IT-remote-access/internal/LSE-VPN) and use SSH as normal.
